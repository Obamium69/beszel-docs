# Beszel documentation website

Documentation for [Beszel](https://github.com/beszel/beszel), powered by [VitePress](https://vitepress.dev/).

### Install dependencies

```bash
bun i
```

```bash
pnpm i
```

### Start dev server

```bash
bun dev
```

```bash
pnpm dev
```

### Build

```bash
bun run build
```

```bash
pnpm run build
```
